[번개케어]
번개장터 | 2021.12 ~ 2022.06

개요
  - 기존의 번개장터, 중고 커머스의 C2C 시장에서 볼 수 없던 C2B2C 중고상품 캐어 시스템 개발
  - 상품등록간에 상품명/카테고리 기반의 번개케어가 가능한 모델 추천

환경 및 기술
  - springboot 2.7.x, kotlin 1.7.x, mysql 5.7.x
  - jpa, redis cluster, AWS-kinesis, airflow

경험내용
  - 번개케어가 가능한 카테고리 및 브랜드를 식별할 수 있도록 번개케어 타겟 관리 설계 및 API 개발
  - 번개케어 상품 상태 변경을 위해 상품 및 주문, 결제, 배송 시스템의 kinesis 이벤트 송/수신 처리
  - airflow, batch task 등을 활용하여 번개케어 거래 현황 발송 시스템 개발
  - elasticsearch query를 적절히 적용하여, 상품명/카테고리 기반 모델 추천 API 개발(비정상 상품명 제외)

성과
  - 서비스 오픈 당시 22년4월 월 430여 건 거래 → 22년12월 월 8000여 건의 유료 거래 달성
  - 검수 리드타임을 단축하기 위해 운영 지원 시스템을 개발하여, 입고 당일 출고율을 10% → 60% 이상으로 개선
  - 상품명/카테고리 기반 모델 추천 정확도 80~90% 이상 달성

  - springframework 4.x, java 1.8, mysql 5.7.x
  - spring batch, mybatis, angular js, redis, memcached

  - 프로모션 정책, 상품, 스케줄링 등 변경이 거의 없는 정보는 memcached를 활용하여 저장
  - 비정상적인 공격과 다중 요청을 방지하기 위해, 스케줄과 사용자 정보에 unique key 설정

성과
  - 일 평균 25~30만 명의 참여자를 대상으로 안정적인 룰렛 프로모션 운영
  - 룰렛 프로모션을 통해 구매효율과 구매건수를 증대시켜, 타임 커머스 운영에 큰 기여
